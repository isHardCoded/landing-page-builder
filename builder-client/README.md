## Installation

Установка проекта

```bash
  git clone https://github.com/isHardCoded/landing-page-builder.git
  cd landing-page-builder
  cd builder-client
```

Установка зависимостей

```bash
  yarn add json-server
```

Запуск клиента

```bash
  yarn dev
```

Запуска сервера

```bash
  yarn server
```
